# customer_segmentation
Customer Segmentation Analysis implementation using RFM (Recency Frequency and Monatery) modeling technique.
 Customer Segmentation: Introduces RFM analysis and K-Means clustering in Python for understanding customer loyalty.
 Data Cleaning: Explains cleaning steps for the Online Retail Dataset, including handling missing values and duplicates.
 RFM Scoring and Clustering: Demonstrates RFM scoring and K-Means clustering to identify loyal customer segments.
